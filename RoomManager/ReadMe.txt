Beschreibung RoomManager
Am Anfang ist man auf der Index-seite, dort kann man entweder einen Key eingeben oder einen Raum reservieren.
Wenn man einen Key eingibt, dann überprüft das Programm, ob es ein PublicKey oder ein PrivateKey ist.
Ist es ein PublicKey, so wird man auf eine Seite geleitet, bei welcher man den Termin ansehen, aber nicht bearbeiten kann.
Wenn es aber ein PrivateKey ist, dann werden alle Informationen aufgelistet, und man kann diese bearbeiten. Ausser den beiden Keys, diese kann man nicht bearbeiten.
Auf allen Seiten kann man auch mit einer Schaltfläche auf die Index-seite zurück.
Wenn man auf der Index-seite auf „Raum reservieren“ drückt, dann kommt man auf eine Seite, auf der man einen neuen Termin erfassen kann.
Wenn man beim Bearbeiten, erfassen oder auch beim Schlüssel eingeben einen nicht akzeptierten Input tätigt, dann erscheint eine hilfreiche Fehlermeldung und man kann es nochmal versuchen
